# Worker 状态表

此文件记录所有工作 Agent 的当前状态。

## 状态说明

- `idle` — 空闲，等待任务
- `working` — 工作中
- `done` — 任务完成
- `error` — 发生错误

> **与 agent-status.md 的关系**：本文件记录 Worker **运行时**细粒度状态（idle/working/done/error），agent-status.md 记录 **全局** 生命周期状态（未启动/活跃/沉睡）。Worker 完成任务后应同时更新两个文件。

## 心跳机制

每个工作 Agent 应在完成任务或重要里程碑后更新此文件。
时间戳格式: `YYYY-MM-DD HH:MM:SS`

---

## 使用说明

1. Worker 启动时，在此表中注册（如果尚未注册）
2. 认领任务后，更新状态为 `working`，记录当前任务
3. 完成任务后，更新状态为 `idle`
4. 遇到问题无法继续时，更新状态为 `error` 并备注原因
