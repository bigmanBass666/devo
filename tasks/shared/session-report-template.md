# 会话报告模板

> 各 Agent 会话结束后按此模板写入 `tasks/shared/session-reports/[角色].md`。
> 普通模式写简版，演练模式写详版。

## 简版（普通模式）

```markdown
## [YYYY-MM-DD HH:MM] [会话目标]

### 执行动作
- [x] 动作1: 描述
- [ ] 动作2: 描述（未完成）

### 改进建议
- [建议描述]

---
```

## 详版（演练模式）

```markdown
## [YYYY-MM-DD HH:MM] [会话目标] 🎭 演练

### 执行动作
- [x] 动作1: 描述
- [ ] 动作2: 描述（未完成）

### 协议合规
- [ ] 角色切换: 是否以正确 Agent 身份执行
- [ ] 铁门协议: 是否只输出"请唤醒 [Agent]"
- [ ] 日志记录: 是否写入对应 .log 文件
- [ ] 状态更新: 是否更新 agent-status.md

### 发现的问题
- [问题描述]（严重程度: P0/P1/P2）

### 改进建议
- [建议描述]

---
```

## 演练模式触发

用户说"唤醒 [Agent名] 演练模式"或"演练模式，唤醒 [Agent名]"时进入演练模式。

## 追加位置

新报告**必须**追加在 session-reports 文件的"报告记录"标题正下方，重置分隔线（`## ⏬ 系统重置分隔线`）之前。

```
## 报告记录

> 以下按时间倒序排列，最新报告在最上方。

← 新报告追加在这里（重置分隔线之前）

---

## ⏬ 系统重置分隔线 — ...

← 不要追加到这里（重置分隔线之后）
```
