# 改进队列

此文件记录 Maintainer 提出的改进建议及其状态。

## 状态说明
- `proposed` — 已提出，待审批
- `approved` — 用户批准，待实施
- `implementing` — 正在实施
- `done` — 已完成
- `rejected` — 用户拒绝

## 格式模板

```markdown
### [IMP-XXX] [标题]
- **优先级**: P0/P1/P2/P3
- **类型**: flow(流程)/tool(工具)/doc(文档)/arch(架构)
- **状态**: proposed/approved/implementing/done/rejected
- **提出时间**: YYYY-MM-DD HH:MM
- **目标**: 改进什么
- **问题**: 解决什么问题（基于日志分析）
- **方案**: 具体怎么做
- **涉及文件**: 需要修改哪些文件
- **预期效果**: 预期改善什么指标
- **实施结果**: （done 后填写）
```
