# 任务分配表

此文件记录 Coordinator 分配给 Worker 的子任务。

## 任务状态

- `pending` — 待认领
- `assigned` — 已分配给 Worker
- `in_progress` — Worker 正在执行
- `completed` — 已完成
- `failed` — 执行失败
- `blocked` — 被阻塞

---

## 使用说明

1. Coordinator 拆分任务后，在此表中创建记录
2. Worker 认领任务后，更新 `分配给` 和 `状态`
3. Worker 完成后，更新 `状态` 为 `completed`，填写 `完成时间`
4. 如果失败，更新 `状态` 为 `failed`，并备注原因
