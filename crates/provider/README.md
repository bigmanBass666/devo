the `provider` crate implements llm invoke sdk, mainly two provider family.
- openai
- anthropic

for openai family, there are two transport: `Legacy Chat Completion` and `New Response API`.
- Chat Completion: https://developers.openai.com/api/reference/chat-completions/overview
- Response: https://developers.openai.com/api/reference/resources/responses

For Anthropic family, has one transport named `Messages`, here is documentation: https://platform.claude.com/docs/en/api/messages
